Thank you for downloading simutrans AI player.

-How to use-
1. Put sqai_koa folder to simutrans/ai folder.
2. Play simutrans.
3. In the Player List dialog (shortcut key: k button), select "Scripted AI's" in any slot.
4. When the check box is turned on, the folder select dialog is activated. Select sqai_koa.

-Caution-
Please use it as a trial run as it may corrupt your save data.
Please use the Japanese forum to report bugs (https://forum.japanese.simutrans.com/index.php?topic=633.msg3243;topicseen#msg3243). 
It would be helpful if you could attach an error dialog to your report.
